# OS-study

## 📁 Repository Structure
본 레포지토리는 스터디 참여자별로 디렉토리를 분리하여 관리합니다.
```
OS-study/
├── minchae/
│   ├── OP_TEE.cpp
│   ├── week02/
│   └── ...
├── gildong/
│   ├── week01/
│   └── ...
```
* 최상위 폴더 아래에 본인의 영문 이름으로 디렉토리 생성
* 그 아래에는 자유롭게 폴더 구성 가능, 파일 이름 또한 자유롭게 작성 (제한 없음)

## ✅ PR 규칙
🔽 PR 제목 형식 
```
Assignment: 과제 제목
```
* 하나의 PR에는 하나의 주제 (과제)만 포함
* 코드 실행 가능 상태 유지
* (선택) 간단한 설명 또는 실행 방법 포함 권장
